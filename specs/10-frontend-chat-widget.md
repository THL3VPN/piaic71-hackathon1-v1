# Feature: Frontend Chat Widget (Reusable)

## Requirements
- Implement chat widget in `/frontend` as a reusable component.
- Widget supports:
  - chat input + responses
  - shows citations
  - error state when backend unavailable
- Widget supports selected-text mode by accepting a `selectedText` prop (optional).

## Acceptance Criteria
- Widget can be embedded into Docusaurus pages.
- Widget can call backend endpoints and render results.
